Before you submit, heres the requirements:

Stack 1:

Have a frontend made in HTML, CSS, and JS with a form, text, and an image!


Stack 2:

Have a backend which serves the frontend, and has 2 API endpoints
implementing the creation and reading of something!


Stack 3:

Have a full stack app where the frontend talks to the backend,
the backend provides interactivity for the frontend
and you've customized your app!


What counts as customized?

If you're following the guide, make the website look unique (make it your own!)
and add at least one extra API endpoint on the backend

Otherwise, use common sense! Don't copy other people, and don't copy the guide!

What is pancakes?

Pancakes is a YSWS where you build a full stack app by going through "stacks", and get rewarded with pancakes!


Stack 1:

Have a frontend with a form, some text, and an image!


Stack 2:

Have a backend with two API endpoints: one to create something and another to delete something!


Stack 3:

Have a website which interfaces with the API endpoints along with some customizations, like additional CSS
